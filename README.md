## 🚀 Project Overview
As a selected **Student Digital Ambassador**, this repository serves as my foundational Digital Literacy Portfolio. The goal of this project is to demonstrate proficiency in professional digital presence, online collaboration, ethical communication, and cybersecurity awareness.

Each folder in this repository corresponds to a core module of the Digital Literacy course:
* **Task 1:** Awareness Infographic (Visual Literacy)
* **Task 2:** Digital Portfolio (Professional Presence)
* **Task 3:** Platforms & Collaboration (Technical Skills)
* **Task 4:** Email Etiquette (Professionalism)
* **Task 5:** Cybercrime Case Study (Safety & Ethics)
